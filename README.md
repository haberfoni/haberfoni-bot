# Haberfoni Bot

Bu servis, haber kaynaklarından otomatik haber çekmek için tasarlanmıştır.

## Kurulum
1. `npm install` ile bağımlılıkları yükleyin.
2. `.env` dosyasındaki `DATABASE_URL` kısmını veritabanı bilgilerinizle güncelleyin.
3. `node index.js` ile botu başlatın.

## Docker ile Çalıştırma
`docker compose up -d`
